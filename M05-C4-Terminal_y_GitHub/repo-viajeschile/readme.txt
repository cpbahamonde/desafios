** Descripción **
La agencia Viajes Chile solicita la creación de su landing page, que tenga un diseño atractivo
e innovador, utilizando los conocimientos adquiridos a través de las distintas
unidades, de acuerdo a la estructura definida en documento: "Prueba - Viajes Chile.pdf"

La página contiene las siguientes secciones:

    ● Una barra de navegación fija en la parte superior de la pantalla, con el logo y links a
    diferentes secciones de la página, haciendo un smooth scroll.
        - SECCION "header-navbar" 

    ● Un carousel que muestra las imágenes destacadas del sitio.
        - SECCION "header-Carousel" 

    ● Una sección de presentación, utilizando favicons y 3 párrafos (desapareciendo en
    tamaños pequeños de pantalla).
        - SECCION "main-QSomos"

    ● Una sección de destacados, que muestre 4 cards (tarjetas) con la imagen e
    información asociada.
        - SECCION "main-Destacados" 

    ● Una sección de formulario de contacto.
        - SECCION "main-Contacto"

    ● Una sección footer con links a las redes sociales.
        - SECCION "footer"

** Requerimientos **
1. Se construye la estructura de la página usando HTML (archivo "index.html").
    ● Se construyó la estructura de un archivo HTML, utilizando etiquetas 
        semánticas para definir las distintas secciones de la página.
    ● Se realiza la carga de archivos y organización del directorio.

2. Se compone visualmente el documento HTML con propiedades CSS (archivo "assets/style.ccs").
    ● Se aplica sintaxis de CSS utilizando selectores y clases, definiendo el aspecto visual del documento.
    ● Se utilizan recursos externos con acceso CDN como Bootstrap, Font Awesome y Google Fonts, para definir
    estilos de fuente.
    ● El código se encuentra ordenado, documentado e indentado.

3. Se emplearon elementos de Bootstrap en el documento HTML.
    ● Se realizar integración del CDN de Bootstrap en sección "head".
    ● Se emplearon componentes de Bootstrap (excluyendo aquellos con JavaScript), haciendo uso de la grilla 
        y de clases utilitarias, , tales como navbar, rows y forms..

4. Se agregaron componentes que utilizan JavaScript (archivo "assets/app.js").
    ● Se utilizó sintaxis de JavaScript.
    ● Se aplicaron dos o más componentes de Bootstrap JS (carousel, tooltips, entre otros).
    ● El código se encuentra ordenado y documentado, incluyendo la fuente de origen.
